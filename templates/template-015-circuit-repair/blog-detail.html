// 手機泡水怎麼辦？急救指南

2026-03-15 | 作者：陳柏翰 | 閱讀時間：5 分鐘

手機不慎掉入水中是許多人的噩夢。根據我們的維修統計，泡水機占所有維修案例的 15%，而正確的緊急處理方式能將修復成功率從 40% 提升到 80% 以上。以下是你需要知道的急救步驟。

// 黃金急救時間

手機泡水後的前 30 分鐘是黃金搶救期。水分在這段時間內可能尚未完全滲入主機板核心區域，如果能正確處理，受損程度可以大幅降低。

// 立即要做的事

  1. 立刻撈出並關機 — 這是最重要的一步。如果手機還亮著，長按電源鍵強制關機。通電狀態下水分會造成短路。
  2. 取出 SIM 卡和記憶卡 — 打開卡槽，取出所有卡片，讓卡槽開口幫助水分排出。
  3. 用乾布擦拭外部 — 輕輕擦去外部所有可見的水分，特別是充電孔、耳機孔、喇叭孔。
  4. 自然風乾 — 將手機放在乾燥通風處，充電孔朝下，讓水分自然流出。

⚠️ 絕對不要做的事

• 不要用吹風機熱風吹（會將水分推入更深處，且高溫損壞零件）
• 不要甩手機（水分會擴散到更多區域）
• 不要嘗試充電（通電會立即短路）
• 不要放入米中（米粒粉塵會堵塞接口，且效果極微）

// 什麼時候該送修？

如果手機掉入水中超過 5 秒、掉入非清水（海水、飲料、泳池水）、或關機後嘗試開機出現異常，建議立刻送修。不要等到「自然好了」再說，因為水分中的礦物質會持續腐蝕主機板。

💡 專業建議

將手機用衛生紙包裹，放入密封袋中，盡快帶到我們店裡。我們有專業的超音波清洗設備和無水酒精清潔流程，能在水分造成永久損壞前進行搶救。緊急送修請直接到店，無需預約。

// 專業維修流程

當泡水機送到我們手上，我們會：

  • 完全拆機，分離所有模組
  • 使用超音波清洗機清潔主機板
  • 無水酒精清除殘留水分與礦物質
  • 紅外線烘乾確保完全乾燥
  • 顯微鏡檢查是否有腐蝕或短路點
  • 修復受損的焊點或更換損壞的零件

// 預防勝於治療

雖然現代手機大多有 IP68 防水等級，但防水能力會隨使用時間遞減。建議：

  • 避免帶手機進浴室（蒸氣也會侵入）
  • 使用防水手機袋在水邊活動
  • 摔過的手機防水性能會大幅降低
  • 定期備份重要資料到雲端

如果不幸遇到手機泡水的緊急狀況，歡迎隨時帶到 Circuit 電子維修，我們的技師團隊有豐富的泡水機搶救經驗，會盡全力幫您搶救愛機與珍貴資料。
