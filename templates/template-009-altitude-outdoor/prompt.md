### 技術規格

- **HTML5 語義標籤**：`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- **全部 CSS inline**：每頁 `<style>` 標籤內，不引用外部 .css 檔案
- **Google Fonts**：透過 `<link>` 引入 Oswald + Noto Sans TC
- **RWD 斷點**：768px（平板）/ 480px（手機）
- **手機選單**：從**右側**滑入，背景 `var(--surface)`，邊框 `var(--border)`，文字 `var(--text)`，Oswald 大字（2rem）
- **Hero 視差效果**：`background-attachment: fixed` 製作偽 parallax
- **CTA 按鈕**：主按鈕 — 綠色實心，uppercase Oswald，letter-spacing: 0.1em；次按鈕 — 透明背景 + 白色邊框
- **山形分隔線**：使用 CSS `clip-path: polygon(0 0, 100% 0, 100% 80%, 50% 100%, 0 80%)` 製作山峰效果
- **數字計數器**：JS `IntersectionObserver` 觸發數字動畫（從 0 滾動至目標值）
