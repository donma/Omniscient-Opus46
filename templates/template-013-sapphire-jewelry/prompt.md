### 技術要求

- 純 HTML5 + CSS3 + 原生 JavaScript，不使用任何框架或函式庫
- 所有頁面共用同一個導航列與頁尾（以 JavaScript 動態插入或各頁面複製）
- 手機版響應式設計，斷點：768px（平板）、480px（手機）
- 手機版 hamburger 選單從右側滑入，帶有 overlay 遮罩
- 滾動時導航列加深背景色（加入 scrolled class）
- 卡片與區塊使用 Intersection Observer API 實現入場動畫（fade-up）
- 所有內部連結相互對應，不出現死連結

請生成完整的 12 個 HTML 檔案，每個檔案都包含完整的 CSS（內嵌 `<style>` 或共用 style.css）與 JavaScript，可以直接在瀏覽器開啟使用。
