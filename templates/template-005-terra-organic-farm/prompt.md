### 品牌設定

**品牌名稱：** Terra 有機農場（Terra Organic Farm）
**產業：** 有機農業／CSA 社區支持農業／在地食材直送
**地點：** 宜蘭縣壯圍鄉大福路二段 15 號（本農場）；台北直送服務範圍覆蓋大台北地區
**創立年份：** 1978 年（陳家農場創立），2010 年轉型有機認證
**三代農家：**
- 第一代：陳水木（阿公），1978年於宜蘭購地，種植水稻、蔬菜
- 第二代：陳建全（阿爸），1998年引入溫室技術，擴充品項至百種蔬果
- 第三代：陳品妤（現任負責人），台大農藝系畢，2010年返鄉推動有機轉型並創立 Terra 品牌，同年取得有機認證，2015年建立 CSA 訂閱制直送系統
**品牌理念：** 「土地是我們的語言，餐桌是我們的答案」——三代農耕智慧結合現代有機農業知識，拒絕農藥與化肥，以最誠實的方式種出最安全的食物，直送到每一個家庭的餐桌。
**認證：** 農委會有機農產品認證、慈心有機認證、ISO 22000 食品安全管理系統
**CSA 訂戶數：** 現有 380 個家庭訂戶（台北 280 個、宜蘭在地 100 個）
**農場規模：** 15 公頃，含水稻田 6 公頃、蔬菜溫室 4.5 公頃、果樹區 2.5 公頃、體驗農園 2 公頃
**目標客群：** 注重食安的都市家庭（有幼兒的父母、素食者、健康管理需求者），年齡 28–50 歲，願意支付溢價購買有機食材。

### 設計規格

**色彩系統 (CSS Variables)：**
```css
:root {
  --bg: #faf8f4;           /* 主背景：溫暖米白，如米糠色 */
  --white: #ffffff;        /* 純白：卡片背景 */
  --green: #5c7a3a;        /* 主綠色：葉片深綠，品牌主色 */
  --green-light: #7a9c50;  /* 淺綠：hover 狀態 */
  --green-pale: #edf2e5;   /* 極淡綠：背景點綴 */
  --brown: #8b6914;        /* 主棕色：土壤色，次要強調 */
  --brown-light: #c4a35a;  /* 淺棕：按鈕、標題裝飾 */
  --text: #2d2a26;         /* 主文字：深咖啡黑 */
  --muted: #6a6050;        /* 次要文字：大地灰棕 */
  --border: #e0d8c8;       /* 邊框：米黃色 */
  --cream: #f5f0e8;        /* 區塊背景：奶油色 */
}
```

**字體 (Google Fonts)：**
- 標題字體：`Playfair Display`，weight 400、700（帶有優雅的農莊書刊質感，用於所有 H1–H3）
- 副標題：`Playfair Display`，weight 400 italic（引言、農場故事段落）
- 內文字體：`Noto Sans TC`，weight 300、400（用於段落、導覽、按鈕）
- Google Fonts 引入語法：
  ```html
  <link href="https://fonts.googleapis.com/css2?family=Playfair+Display:ital,wght@0,400;0,700;1,400&family=Noto+Sans+TC:wght@300;400;500&display=swap" rel="stylesheet">
  ```

**設計風格：**
- **整體氛圍：** 如翻閱一本精緻的農場生活誌，溫潤、真實、有機的質感。沒有過度設計感，像一位農夫的手寫筆記本轉化為網站。大地色系，大量自然採光感圖片。
- **Hero 版型：** 敘事捲動型（Storytelling Scroll）——Hero 區使用全寬農場風景大圖，文字覆蓋在圖片上，帶有半透明深色遮罩。往下捲動有故事感的 section 交錯排列。
- **卡片設計：** 米白背景（`var(--cream)`），`border: 1px solid var(--border)`，`border-radius: 8px`，`box-shadow: 0 2px 16px rgba(0,0,0,0.04)`，帶有一種手工質感。圖片上緣略帶圓角，下方留白文字區。
- **按鈕設計：** 主按鈕為深綠色（`background: var(--green)`），白色文字，`border-radius: 4px`，hover 時稍亮（`var(--green-light)`）。次要按鈕為棕色 outline（`border: 2px solid var(--brown)`，`color: var(--brown)`）。按鈕字體 Noto Sans TC 400，`letter-spacing: 0.05em`。
- **裝飾元素：** 章節標題旁使用手繪風葉片圖示（Unicode 葉型字符或 SVG），分隔線使用棕色點線（`border-top: 1px dashed var(--border)`）。
- **圖片處理：** 暖色調優先（`filter: saturate(1.1) brightness(1.02)`），帶有自然採光感。比例：農產品圖片 1:1（方型），農場風景 16:9，人物故事圖 3:4。
- **Wide Images：** 使用 `width: 100vw; margin-left: calc(-50vw + 50%)`（突破容器的全寬圖片）在故事敘述節點插入全寬農場圖，增加沉浸感。
- **間距哲學：** section padding 上下各 `80px`，強調呼吸感。

**Border Radius：** `8px`（卡片）、`4px`（按鈕）、`50%`（農夫頭像）
**手機選單方向：** 左側滑入（`left: 0`）

**語調：** 真誠、溫暖、誠實。像農夫寫給城市消費者的信，沒有行銷腔，充滿具體細節（天氣、土壤、蟲咬痕跡）。承認不完美（有機蔬菜不總是外觀完美），以誠實建立信任。
