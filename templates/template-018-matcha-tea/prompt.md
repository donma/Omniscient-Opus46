- 與其他流派差異：裏千家重視「生活中的茶」，不過度形式化，親近一般大眾
- 全球推廣：裏千家在全球 60 個國家設有研究會

**吳宛庭的學習歷程：**

**「一期一會」的意義（300字說明）：**

#### 4. menu-detail.html（濃茶詳細頁）

茶品名稱：「濃茶（抹茶）」日文：「濃茶 / こいちゃ」

完整介紹（分區塊）：

**什麼是濃茶？**
「濃茶是抹茶中最頂級的沖泡方式，使用量約為薄茶的三倍，以少量熱水調和成質感如漿糊般濃稠的茶液。喝濃茶是茶道正式儀式的核心，茶色深翠，香氣馥郁，入口先苦後甘，是一種只有在安靜中才能完全體會的滋味。」

**茶粉來源：**
「我們使用的抹茶粉，來自京都宇治 若竹製茶所 的百年茶園。每年四月，茶農為最高級的玉露和濃茶茶園蓋上遮光網罩，讓茶芽在減少日照的環境下生長 20 天，此時茶胺酸大量累積，形成濃厚的鮮味（旨味）與深沉的翠綠色。」

**沖泡規格：**
茶粉量：3.75 克（約 2 茶匙）
水溫：80°C（非沸騰，避免破壞茶胺酸）

茶碗：使用 Matcha 茶道空間特訂的 信樂燒 茶碗
沖泡方式：以天目沖法，遵循裏千家傳統點前步驟

**與薄茶的差異：**

|--|------|------|
| 茶粉量 | 3.75g | 1.5g |
| 質地 | 濃稠如漿 | 泡沫輕盈 |
| 滋味 | 深厚、先苦後甘 | 清新、帶甜 |
| 儀式性 | 高（茶道正式茶） | 中（日常品茶） |
| 建議搭配 | 主菓子（練切） | 干菓子 |
