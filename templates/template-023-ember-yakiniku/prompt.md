#### 2. about.html — 品牌故事

**主廚故事 — 林俊傑（400字）：**
2014年，林俊傑帶著一個信念飛往東京——他要找到燒肉最本質的味道。他進入銀座三丁目的「炭火の真」，從最基礎的炭火管理學起：炭的溫度、火焰的高度、翻轉的時機，每一個動作都需要精確到秒。師父田中一郎告訴他：「備長炭不是工具，是料理的靈魂。」

五年後，林俊傑帶著在東京習得的技藝回到台北。他選擇大安區一個安靜的巷弄，打造了一間只有28席的燒肉餐廳，命名為 Ember——那一顆永遠燃燒的炭火餘燼。

**備長炭哲學段落（200字）：**
世界上的炭火燒肉，99%使用瓦斯爐模擬炭火效果。Ember 選擇備長炭，是因為備長炭燃燒時，遠紅外線穿透肉的纖維，由內而外均勻加熱，鎖住肉汁的同時，炭的香氣滲入每一絲肌理——這是任何瓦斯設備無法複製的味道。

**品質承諾三段：**
- 每週從日本空運宮崎縣A5等級和牛，附有品種血統認證書
- 所有肉品由持有「和牛分切師」認證的師傅現場處理
- 不使用任何人工醃料，僅以天然鹽、醬汁或主廚自製熟成奶油調味

- 純 HTML5 + CSS3 + 原生 JavaScript（無框架依賴）
- 全站深色主題，嚴禁大面積白色背景
- 所有頁面共用同一個 `style.css` 與 `nav.js`
- 圖片使用深紅色漸層佔位符（模擬炭火光）
- 完全 RWD 響應式設計
- 手機版導覽列從**左側**滑入，極深暗紅背景
- 菜單頁使用仿奢華印刷風格（黑底金字邊框裝飾）
- `--fire` 色的 glow 效果用於 hover 狀態：`box-shadow: 0 0 15px rgba(230,57,70,0.5)`
- 食材照片使用高對比度濾鏡（`filter: contrast(1.1) brightness(0.9)`）
- 訂位表單的送出按鈕使用炭火橙漸層（`background: linear-gradient(135deg, #ff6b35, #e63946)`）
- 所有 section 標題使用 Oswald 全大寫字母

請按照以上所有規格生成完整的12個 HTML 頁面，每個頁面都需要有完整的 `<html><head><body>` 結構，共用導覽列與頁尾，確保所有內部連結正確對應，並且全站維持頂級燒肉餐廳的戲劇感與食慾誘惑的視覺美學。
