- 標題字體：`Noto Serif TC`，使用 weight 400、500、700（用於所有 h1–h4 及品牌 logo）
- 內文字體：`Noto Sans TC`，使用 weight 300、400、500（用於段落、導覽、按鈕）

  ```html
  <link href="https://fonts.googleapis.com/css2?family=Noto+Serif+TC:wght@400;500;700&family=Noto+Sans+TC:wght@300;400;500&display=swap" rel="stylesheet">
  ```

- **整體氛圍：** 奢華輕熟女風格，如同高端日系美容雜誌的排版質感。大量留白，節制用色，金色點綴。

- **分隔裝飾：** 章節標題下方使用細金線裝飾（`width: 48px; height: 2px; background: var(--gold)`，置中）。

**Border Radius：** `12px`（卡片）、`4px`（按鈕）、`50%`（頭像圓形）
**手機選單方向：** 右側滑入（`right: 0`）

### 技術規格

- 純 HTML + inline `<style>` 標籤，**不使用**任何外部 CSS 檔案
- 使用 Google Fonts（`<link>` 標籤引入，放在 `<head>` 內）

- RWD 響應式設計，主要 breakpoint：`768px`
- 手機版漢堡選單（三條橫線圖示，純 CSS + JavaScript toggle，點擊漢堡圖示時從右側滑入全屏選單）
- 點擊選單外部區域或關閉按鈕可收合選單
- 所有文字內容使用**繁體中文**
- 每頁 header 包含：左側品牌 logo（文字型，Aurora 美學沙龍）+ 右側桌面版導覽列 + 手機版漢堡按鈕
- Header 背景：白色，底部 `border-bottom: 1px solid var(--border)`，`position: sticky; top: 0; z-index: 100`
- 每頁 footer 包含：品牌介紹短文、導覽連結、聯絡資訊（地址/電話/LINE）、營業時間、社群圖示、版權聲明

### 品牌文案風格

**語調：** 溫柔而有力，專業中帶有親近感。避免過度浮誇或醫療廣告式措辭。不使用驚嘆號堆疊，用句點結尾表現沉穩氣質。

**標語範例：**
- 「喚醒肌膚本質光采，從一次療程開始。」
- 「科學，是我們的語言；草本，是我們的溫度。」
- 「不是為了讓你不同，而是讓你更像自己。」
- 「每一位進入 Aurora 的女性，都值得被認真對待。」

**文案節奏：** 短句配長句，節奏有韻律感。例：「我們不追求速效。我們追求的，是你三個月後在鏡子前的那個微笑。」

**忌用詞：** 美白（改用「提亮」「均勻膚色」）、療效（改用「改善」「幫助」）、保證（改用「我們相信」「大多數顧客反映」）
