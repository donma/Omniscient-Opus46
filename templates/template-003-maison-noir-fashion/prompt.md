### 設計規格

**色彩系統 (CSS Variables)：**
```css
:root {
  --bg: #ffffff;          /* 主背景：純白 */
  --off-white: #f8f7f5;   /* 次背景：略帶溫度的白 */
  --black: #1a1a1a;       /* 主黑：標題、線條 */
  --muted: #6b6b6b;       /* 中性灰：次要文字 */
  --border: #e0ddd8;      /* 邊框：極淡灰米色 */
  --accent: #1a1a1a;      /* 強調色即黑（品牌拒絕彩色） */
}
```

**字體 (Google Fonts)：**
- 標題字體：`Cormorant Garamond`，weight 300、400、600，**全面使用 italic 斜體**（這是品牌核心美學標誌）
- 內文字體：`Noto Sans TC`，weight 300、400（極細為主，傳遞輕盈感）
- 英文 body 備用：`Cormorant Garamond` 400 regular（非斜體，用於引言）
- Google Fonts 引入語法：
  ```html
  <link href="https://fonts.googleapis.com/css2?family=Cormorant+Garamond:ital,wght@0,300;0,400;0,600;1,300;1,400;1,600&family=Noto+Sans+TC:wght@300;400&display=swap" rel="stylesheet">
  ```

**設計風格：**
- **整體氛圍：** 如同時尚雜誌《032c》或《System Magazine》的頁面——極簡但不空洞，排版不對稱，文字與圖片之間存在刻意的張力感。
- **版型：** 編輯型雜誌風格（Editorial Magazine）。大量不對稱排版：左欄文字極窄（30–40%），右欄圖片極寬（60–70%）；或反向。圖片與文字之間有大量留白。
- **Hero 版型：** 不使用傳統居中 Hero。改用「兩欄編輯版型」：左半純白背景放極簡文字，右半滿版時裝攝影（高反差、藝術感強的模特兒圖）。
- **卡片設計：** 無圓角（`border-radius: 0`），無陰影，邊框使用極細黑線 `1px solid var(--black)` 或完全無邊框。商品圖片比例 2:3（直式）。
- **按鈕設計：** 反轉色按鈕——黑色背景白色文字（`background: var(--black); color: white`），或完全無背景只有底線（`border-bottom: 1px solid var(--black)`，`background: none`）。字距超寬：`letter-spacing: 0.2em`，字體全大寫（`text-transform: uppercase`），Noto Sans TC 300。
- **分隔線：** 只使用 `1px solid var(--border)` 的細橫線，或完全不用分隔線而以空白分隔章節。
- **間距哲學：** 極大留白。section padding 上下各 `120px`（手機 `60px`）。圖片與文字組件之間刻意使用非等比的不對稱間距，製造緊張感。
- **圖片處理：** 黑白或低飽和度彩色（使用 CSS `filter: grayscale(0.3) contrast(1.1)`），比例嚴格統一。
- **文字排版特殊處理：** 某些大標題使用極小字（`font-size: 11px; letter-spacing: 0.4em; text-transform: uppercase`）作為章節標籤，與其下方的斜體大標題形成強烈對比。

**Border Radius：** `0`（全站無圓角，表達當代性與建築感）
**手機選單方向：** 右側滑入（`right: 0`）
