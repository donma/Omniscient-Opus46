### 設計規格

- **主題**：Light / 純白極簡
- **色彩變數**：
  - `--bg: #ffffff`（主背景）
  - `--off-white: #f9f9f7`（次背景、卡片底色）
  - `--black: #111111`（主文字、標題）
  - `--muted: #777777`（次要文字、標籤）
  - `--border: #e5e5e5`（分隔線、邊框）
  - `--accent: #111111`（按鈕、強調元素，與主黑一致）
- **字體**：
  - 標題：`"Space Grotesk"` (Google Fonts，weights: 300, 400, 500, 700)
  - 內文：`"Noto Sans TC"` (Google Fonts，weights: 300, 400, 500)
  - 藝術品標籤/元數據：letter-spacing: 0.15em，font-size: 0.75rem，text-transform: uppercase
- **排版哲學**：藝術呈現優先，設計退居其次。大量留白，sharp edges（border-radius 最多 2px，多數元素無圓角），極細邊框線條
- **圖片處理**：所有藝術品圖片使用 `object-fit: cover`，以 `background-color: #f0eeec` 佔位。實際開發時以真實藝術品圖片替換
- **Masonry Grid**：使用 CSS columns 或 CSS Grid auto-fill 實作瀑布流排版，桌面版 3–4 欄，平板 2 欄，手機 1–2 欄

### 技術規格

- **HTML5 語義標籤**：`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<footer>`, `<figure>`, `<figcaption>`
- **全部 CSS inline**：每頁 `<style>` 標籤內，不引用外部 .css 檔案
- **Google Fonts**：透過 `<link>` 引入 Space Grotesk + Noto Sans TC
- **RWD 斷點**：768px（平板）/ 480px（手機）
- **手機選單**：從**左側**滑入，使用純 CSS checkbox hack 或 vanilla JavaScript toggle，覆蓋全螢幕，背景 `rgba(17,17,17,0.95)`，文字白色，超大字級（2rem）
- **Hover 效果**：連結底線從左展開動畫（`transform: scaleX`），圖片 hover 輕微 scale(1.02) 帶 `transition: 0.4s ease`
- **頁面過渡**：`<body>` 淡入動畫（`opacity 0 → 1, 0.3s`）
- **無障礙**：所有圖片有 `alt` 描述，focus 狀態可見，色彩對比符合 WCAG AA
