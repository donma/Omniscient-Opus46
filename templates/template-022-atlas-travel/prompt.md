### 技術規格

- 純 HTML5 + CSS3 + 原生 JavaScript（無框架依賴）
- 所有頁面共用同一個 `style.css` 與 `nav.js`
- 圖片使用藍色漸層佔位符（模擬天空/海洋感）
- 完全 RWD 響應式設計
- 首頁 Hero 使用 `background-attachment: fixed` parallax 效果
- 手機版導覽列從右側滑入，白色背景
- 行程卡片的難度標籤使用色彩編碼：輕鬆=綠、中等=橙、挑戰=紅
- 行程詳情頁有黏性側邊欄（sticky sidebar）顯示價格與預訂按鈕
- 所有外部連結（媒體報導）在新分頁開啟
- 滾動時導覽列背景從透明漸變為白色（`scroll` event + CSS transition）

請按照以上所有規格生成完整的12個 HTML 頁面，每個頁面都需要有完整的 `<html><head><body>` 結構，共用導覽列與頁尾，確保所有內部連結正確對應，並且全站維持專業精品旅遊品牌的視覺與文案調性。
