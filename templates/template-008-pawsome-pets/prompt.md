### 技術規格

- **HTML5 語義標籤**：`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- **全部 CSS inline**：每頁 `<style>` 標籤內，不引用外部 .css 檔案
- **Google Fonts**：透過 `<link>` 引入 Quicksand + Noto Sans TC
- **RWD 斷點**：768px（平板）/ 480px（手機）
- **手機選單**：從**右側**滑入，背景使用 `--gradient` 漸層，文字白色，圓角 icon，動畫 `transform: translateX`
- **Hover 效果**：按鈕 hover 輕微上移 `translateY(-2px)` + 陰影加深；卡片 hover `scale(1.02)` + 陰影
- **背景 blob 裝飾**：在關鍵 section 背景加入 CSS `border-radius: 40% 60% 70% 30% / 40% 50% 60% 50%` 的粉紫漸層圓形，`opacity: 0.08`，`position: absolute`，`z-index: 0`
- **無障礙**：所有互動元素有 focus 樣式，色彩對比符合 WCAG AA
