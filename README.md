## 快速開始

直接用瀏覽器開啟根目錄的 `index.html` 即可瀏覽所有模板：


- **搜尋**：輸入關鍵字篩選模板
- **篩選**：按主題（Light/Dark）或產業分類
- **預覽**：點擊模板卡片進入詳情頁，可 iframe 預覽每個頁面
- **下載提示詞**：詳情頁提供「下載 AI 生成提示詞」按鈕

---

## 如何使用 AI 提示詞重新生成模板

每套模板目錄內均附有 `prompt.md` 檔案，內含完整的 AI 生成提示詞。

**步驟：**

1. 在模板詳情頁點擊「⬇ 下載 AI 生成提示詞 (prompt.md)」
2. 用文字編輯器開啟下載的 `prompt.md`
3. 複製「完整提示詞」區塊的全部內容
4. 貼入 Claude、ChatGPT 或任何 AI 助理
5. AI 將依照提示詞生成完整的網站 HTML 檔案

> 提示詞包含：品牌設定、色彩系統、字體規格、所有頁面詳細內容要求、技術規格與文案風格指引。

## 技術規格

- **純靜態 HTML**：所有頁面使用 inline `<style>`，零外部 CSS 依賴
- **字體**：Google Fonts CDN 引入，各模板使用不同字體組合
- **圖片**：全部使用 Unsplash 真實圖片 URL
- **RWD**：768px breakpoint，手機版漢堡選單（左側或右側）
- **語言**：全繁體中文介面與內容
- **無框架**：不依賴 Bootstrap、Tailwind 或任何 CSS 框架
